# ImageBatcher Pro 專案介紹

這是一套基於 Python 開發的批量圖片與影片處理工具，結合了高效的後端邏輯與現代化的圖形介面。

---

## 🛠️ Python 技術亮點

1. **多執行緒非阻塞設計 (Multi-threading)**:
   - 使用 `threading` 模組將耗時的圖片處理任務移至背景執行，確保 GUI 介面在處理大量檔案時依然流暢，不會出現「未回應」的情況。

2. **高效能圖片處理算法**:
   - 整合 `Pillow (PIL)` 函式庫，並採用高品質的 `LANCZOS` 重採樣算法，在縮放圖片時能保持極佳的邊緣平滑度。

3. **互動式 Canvas 畫布**:
   - 基於 `tkinter.Canvas` 實現了互動式的裁剪與縮放功能，支援滑鼠拖曳、即時座標計算與視覺化預覽。

4. **跨多媒體支援**:
   - 整合 `OpenCV` (cv2) 進行影片流處理，實現精確的影格定位與高品質的畫面擷取。

5. **現代化佈景主題 (Themed UI)**:
   - 透過 `ttkthemes` 引入現代化美學設計（如 Arc 主題），擺脫傳統 tkinter 的過時感。

---

## 📦 技術套件與模組一覽表

### 1. 外部第三方套件 (需安裝)
這些套件負責核心的影像與影片處理功能。

| 套件名稱 | 在程式碼中的名稱 | 主要用途 |
| :--- | :--- | :--- |

### 2. Python 內建模組 (免安裝)
這些是 Python 標準庫提供的功能，支撐起程式的運作骨架。

## 📂 模組架構說明

專案採用模組化設計，職責分明，易於維護：

- **`main.py`**: 
  - 應用程式入口，負責初始化核心視窗並啟動事件迴圈。
- **`gui.py`**: 
  - 介面邏輯中心。包含所有分頁（壓縮、縮放、裁剪等）的佈局規劃、元件交互以及使用者事件處理。
- **`image_processor.py`**: 
  - 圖片處理核心。封裝了批量轉換、壓縮算法、縮放運算等底層圖片處理邏輯。
- **`video_processor.py`**: 
  - 影片處理解析器。負責影片檔案的加載、影格索引定位以及將影片訊號轉換為圖片物件。
- **`conversion_handler.py`**: 
  - 併發控制模組。負責排程處理任務，協調後端處理核心與 GUI 進度回傳。

## ⚙️ 3. 環境設定與安裝說明

為了確保專案順利執行，請參考以下步驟進行環境設定：

### 1. 安裝必要套件
本專案依賴於 `Pillow` (圖片處理)、`opencv-python` (影片處理) 以及 `ttkthemes` (介面主題)。
請在終端機執行以下指令：
```bash
pip install -r requirements.txt
```

### 2. 資料夾結構需求
請確保執行時專案目錄完整，包含所有 `.py` 檔案，以便模組正確偵測。

### 3. 執行程式
在終端機進入專案目錄後，執行 `main.py`：
```bash
python main.py
```

### 4. 注意事項
- 處理大量圖片時，請確保目標磁碟空間充足。
- 影片截圖功能支援常見的 `.mp4`, `.avi` 等格式。

# ImageBatcher Pro 功能介紹

這是一套功能強大且簡單易用的批量圖片處理工具，旨在幫助您快速完成各種常見的圖片與影片編輯任務。

## 🚀 核心功能概覽

### 1. 📦 壓縮圖片文檔 (Compress Image)
- **目標**：縮小檔案體積，節省儲存空間。
- **支援格式**：JPG, PNG, SVG, GIF。
- **特色**：在儘可能保持最佳畫質的同時，最大程度地減少檔案大小。

### 2. 🔧 調整圖片的大小 (Resize Image)
- **彈性調整**：支援透過「像素 (Pixels)」或「百分比 (Percentage)」定義尺寸。
- **批量處理**：可一次調整多張圖片，適用於固定尺寸輸出或按比例縮放。

### 3. ✂️ 裁剪圖片 (Crop Image)
- **精確裁剪**：提供直覺的拖曳式介面與座標輸入，精確設定裁切範圍。
- **即時預覽**：即時預覽裁切結果，確保輸出符合預期。

### 4. 🔄 轉換至 JPG 文檔 (Convert to JPG)
- **格式轉換**：支援將各種輸入格式（PNG, BMP, WebP 等）快速轉換為標準的 JPG 格式。
- **簡化工作流**：一次處理大量不同格式的圖片，統一輸出格式以便後續使用。

### 5. ↻ 旋轉圖片 (Rotate Image)
- **多角度旋轉**：支援每次 60°, 90° 或 180° 的旋轉操作。
- **預覽功能**：在儲存前可以即時看到旋轉後的視覺效果。

### 6. 🎬 影片截圖 (Video Screenshot)
- **畫面擷取**：加載影片檔案，並從中擷取特定影格。
- **高畫質輸出**：將擷取的影格直接保存為高品質的圖片檔案。
